# Executive Tableau Dashboard

## Business Problem

This dashboard helps retail leadership monitor sales, profitability, customer behavior, shipping performance, discounts, and returns for better business decisions.

## Dataset

Retail sales dataset containing orders, products, customers, shipping, profits, discounts, returns, and campaign information.

## Tableau Workbook

executive_dashboard.twbx

## Calculated Fields

- Profit Margin
- Cost
- Average Order Value
- Return Rate
- Shipping Delay Bucket

## Dashboard Components

- Sales Trend
- Regional Performance
- Category Profitability
- Customer Segment
- Shipping Performance
- Discount vs Profit
- Return Analysis

## Filters

- Region
- Category
- Customer Segment
- Date

## Key Insights

Sales and profit vary by region, discounts reduce profitability in some categories, delayed shipping impacts returns, and customer segments show different purchasing behavior.

## Story Summary

The dashboard highlights strengths, identifies underperforming regions and products, and supports strategic decisions through interactive visualizations.

## Assumptions

- Return Flag indicates returned orders.
- Delivery Days represent shipping duration.
- Order ID uniquely identifies each order.

## Limitations

Campaign effectiveness and customer lifetime value are not analyzed.
